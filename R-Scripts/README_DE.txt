Voraussetzungen:
	* R (https://cran.r-project.org/)
	
Installation:
	* Es werden bestimmte Pakete von R gebraucht, die mittels Installation.R installiert werden können.
	  Das Skript 'Installation.R' muss aber 2-3 Mal ausgeführt werden, bis wirklich alles korrekt installiert ist.
	  Dazu "./R.exe --no-save" im Ordner "bin" aufrufen.
	  Dann "source('<PfadZuInstallation.R>')" (z.B. source('C:/MeinOrdner/Installation.R')) aufrufen.
	  
Nach der Installation muss nur noch das shell-Skript ausgeführt werden.
